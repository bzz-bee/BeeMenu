# BeeMenu
## An app launcher for hyprland, inspired by the Windows 7/Vista Start Menu.
- Coded in Python + Qt
- Easily modify the UI style
- ! Still in testing & development !
### Installation
All you have to do is clone with repo. To launch, run `python3 ~/BeeMenu/beemenu.py` (an alias will be added soon).
#### Add to your hyprland.conf:
  $menu = command python3 ~/BeeMenu/beemenu.py
