# Bee Menu
## An app launcher for Hyprland, inspired by the Windows 7/Vista Start Menu.
- Coded in Python + Qt
- Easily modify the UI style
- Currently in alpha
### Planned features:
- Gradients and color
- Icons
- Mini file explorer (Documents, Pictures, Music, etc.)
### Installation
All you have to do is clone this repo. To launch, run `python3 ~/BeeMenu/beemenu.py` (an alias will be added soon).
#### Add to your hyprland.conf:
$menu = command python3 ~/BeeMenu/beemenu.py
